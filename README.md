# Ihumure - Mental Health Support Platform

Ihumure means "comfort" in Kinyarwanda. We are on a mission to make mental health support accessible, stigma-free, and community-driven for all Rwandans.

## About the Project

Ihumure is a comprehensive mental health platform designed to provide safe, anonymous spaces where Rwandans can:
- Access mental health information and resources
- Connect with peer support communities
- Participate in anonymous chatrooms
- Use AI-powered mental health assistance
- Find wellness resources and events
- Learn about various mental health conditions

## Key Features

- **Anonymous Chatrooms**: Safe spaces for peer support and shared experiences
- **AI Assistant**: Mental health guidance and support powered by AI
- **Wellness Resources**: Comprehensive mental health information and tools
- **Condition Information**: Educational content about various mental health conditions
- **Community Events**: Mental health workshops and support group meetings
- **Professional Resources**: Access to mental health professionals and services

## Our Mission

To provide a safe, anonymous platform where Rwandans can access mental health information, connect with peer support communities, and find resources to support their wellbeing journey.

## Our Vision

A Rwanda where mental health is prioritized, stigma is eliminated, and everyone has access to the support they need to thrive mentally and emotionally.

## Development Setup

To run this project locally:

### Prerequisites
- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd ihumure

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

The development server will start with auto-reloading and an instant preview.

## Technology Stack

This project is built with:

- **Frontend**: React 18 with TypeScript
- **Build Tool**: Vite
- **UI Framework**: shadcn/ui components
- **Styling**: Tailwind CSS
- **State Management**: React Query (TanStack Query)
- **Routing**: React Router DOM
- **Backend**: Supabase (authentication and database)
- **Forms**: React Hook Form with Zod validation
- **Animations**: Framer Motion
- **Icons**: Lucide React
